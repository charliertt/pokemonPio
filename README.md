# pokemonPio
Pokémon MMO-Lite en Vue 3 (Composition API). Implementa un mapa de zonas estilo Duolingo, sistema de lobbies por zona, estado reactivo con ref/reactive, componentes reutilizables y routing dinámico (/map, /zone/:id, /missions, /pvp). Proyecto académico enfocado en UI coherente y arquitectura escalable.
